# Restart-PWSH

## SYNOPSIS
Restarts PowerShell

## SYNTAX

```
Restart-PWSH [<CommonParameters>]
```

## DESCRIPTION
Terminates current PowerShell session and will start a new session.  The alias for this function is 'pwsh'.

## EXAMPLES

### EXAMPLE 1
```
PS C:\> pwsh
```

Terminates current PowerShell session and will start a new session.
